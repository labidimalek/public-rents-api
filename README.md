# Rents Public API

## API endpoint

This course API is available at the following URL:

```plain
https://my-json-server.typicode.com/labidimalek/public-rents-api/rents
```

## Available request methods

* GET: /rents
* GET: /rents/:id
* POST /rents
* PUT /rents/:id
* PATCH /rents/:id
* DELETE /rents/:id
